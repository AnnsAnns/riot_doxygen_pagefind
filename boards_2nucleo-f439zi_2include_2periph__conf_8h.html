ADC configuration

Note that we do not configure all ADC channels, and not in the STM32F439ZI order.

Instead, we just define 6 ADC channels, for the Nucleo Arduino header pins A0-A5 and the internal VBAT channel.

To find appropriate device and channel find in the board manual, table showing pin assignments and information about ADC - a text similar to ADC[X]_IN[Y], where: [X] - describes used device - indexed from 0, for example ADC12_IN10 is device 0 or device 1, [Y] - describes used channel - indexed from 1, for example ADC12_IN10 is channel 10

For STM32F439ZI this information is in MCU datasheet, Table 10, page 53 or in Nucleo-f439ZI board manual, Table 17, page 52.
