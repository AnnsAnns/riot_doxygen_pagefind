Detailed Description

Skald's Eddystone implementation

About

This module allows for creation and advertisement of Eddystone beacons (see https://github.com/google/eddystone).

Implementation state

supported:

  • Eddystone-UID
  • Eddystone-URL

not (yet) supported:

  • Eddystone-TLM
  • Eddystone-EID
