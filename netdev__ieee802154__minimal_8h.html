Detailed Description

Provides basic functionalities to interact with an IEEE 802.15.4 networking device which implements the Netdev - Network Device Driver API.

To use the functionalities, include the module USEMODULE += test_utils_netdev_ieee802154_minimal. The test application should provide:

  • device initialization, via the implementation of netdev_ieee802154_minimal_init_devs
  • number of devices to test, via the definition of NETDEV_IEEE802154_MINIMAL_NUMOF in init_dev.h
