Detailed Description

PLL configuration parameters.

PLL configuration follows the model:

pll_in +----+ vco_in +------------------------------+
--------| /M |----------|\ +----+ vco_out +----+ |
+----+ | --| xN |-------------| /P |--|-- pll_p_out
| +----+ \ +----+ |
| | +----+ |
| ---| /Q |--|-- pll_q_out
| | +----+ |
| | +----+ |
| ---| /R |--|-- pll_r_out
| +----+ |
+------------------------------+

vco_in = pll_in / M; vco_out = vco_in * N; pll_p_out = vco_out / P; pll_q_out = vco_out / Q; pll_r_out = vco_out / R;
