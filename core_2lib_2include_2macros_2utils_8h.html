    ◆ LIMIT

    #define LIMIT ( val,
    low,
    high )
    Value:
    ((val < low) ? low : (val > high) ? high : val)

    Limit a value to an inclusive range.

    If val is > high, high is returned. If val is < low, low is returned. Otherwise, val is returned.

    Note
    This macro evaluate its arguments more than once.
