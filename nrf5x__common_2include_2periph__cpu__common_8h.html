    ◆ GPIO_MODE

    #define GPIO_MODE ( oe,
    ic,
    pr,
    dr )
    Value:
    (oe | (ic << 1) | (pr << 2) | (dr << 8))

    Generate GPIO mode bitfields.

    We use 4 bit to encode the pin mode:

    • bit 0: output enable
    • bit 1: input connect
    • bit 2+3: pull resistor configuration
    • bit 8+9+10: drive configuration
