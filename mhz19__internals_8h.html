MH-Z19 Baud rate

Fixed at 9600 by design.

#define MHZ19_UART_BAUDRATE   9600
 

MH-Z19 Buffer size

A transmission from the MH-Z19 is 9 bytes long: 1 Start byte 1 Sensor number 2 bytes data 4 bytes padding 1 byte checksum

The start byte is not stored because it is not used in the checksum calculation.

#define MHZ19_BUF_SIZE   8
 

MH-Z19 Timeout in milliseconds

20 ms gives a decent margin on top of the UART transmission time.

The datasheet does not specify any timings beside the UART baud rate.

A single byte takes 10 bits effectively: a start bit, 8 bits data and stop bit. 9 bytes are transmitted, thus 10 bits * 9 / 9600bps = 9.3 ms.

#define MHZ19_TIMEOUT_READ   20
 
#define MHZ19_TIMEOUT_CMD   (MHZ19_TIMEOUT_READ * 5)
