LED (on-board) configuration

LED configuration depends on Wemos ESP32-C3 mini board version:

  • v1.0.0 has one LED connected to GPIO7.
  • v2.1.0 has one 24-bit RGB-LED WS2812B connected to GPIO7.

◆ BTN0_PIN

#define BTN0_PIN   GPIO9

Default button GPIO pin definition.

The Wemos ESP32-C3 mini board has a BOOT button connected to GPIO9, which can be used as button during normal operation. Since the GPIO9 pin is pulled up, the button signal is inverted, i.e., pressing the button will give a low signal.
