    A wrapper function to simplify the generation of a hash.

    This is useful for generating sha512_224 for one single buffer in a single step.

    Parameters
    [in]datapointer to the buffer to generate hash from
    [in]lenlength of the buffer
    [out]digestoptional pointer to an array for the result, length must be at least SHA512_224_DIGEST_LENGTH
